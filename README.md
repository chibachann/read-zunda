# read-zunda

Claude Codeがリサーチした内容をドキュメントにまとめ、ずんだもん（VOICEVOX）に読み上げさせるソフトウェアです。

## 概要

1. **リサーチ**: Claude Codeが指定したテーマについてWebリサーチを行い、内容を整理する
2. **ドキュメント生成**: リサーチ結果をMarkdownドキュメントとしてまとめる
3. **音声読み上げ**: VOICEVOXのずんだもんボイスで生成したドキュメントを読み上げる

## 技術スタック（予定）

- **Claude Code** - リサーチ・ドキュメント生成
- **VOICEVOX** - 音声合成エンジン（ずんだもんボイス）
- **Python** - バックエンド処理

## 使い方（予定）

```bash
# テーマを指定してリサーチ＆読み上げ
python main.py --theme "調べたいテーマ"
```
